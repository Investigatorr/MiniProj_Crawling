### 다나와 홈페이지에서 원하는 상품 정보를 크롤링해 쉽게 비교할 수 있는 코드 ###

1. 입력해야 하는 값
	1. 제품 이름 입력
	2. 가격 범위 ( 지불 가능한 최저가격 ~ 최대가격) 설정
	3. 찾고 싶은 Page 지정

2. 크롤링 Flow
	1. 코드 실행
	2. 검색하고 싶은 제품 입력 (selenium)
	3. 자동으로 검색 후 Input받은 가격 범위 입력
	4. 가격별로 정렬 후 구매평가 후기 많은 순으로 정렬
	5. 페이지 넘겨가면서 제목 / 가격 / 세부스펙 정보 파싱(BeautifulSoup)
	6. 긁어온 정보 몽고DB에 저장 (Pymongo / MongoDB)
